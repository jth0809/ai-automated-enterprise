# Infrastructure Rules: Terraform and Ansible

Treat infrastructure exclusively as code.
Abstract complex cloud configurations to provide simple paved paths for developers.
Validate all infrastructure changes automatically to prevent misconfigurations.
Keep state consistent and immutable wherever possible.
STRICT LIMIT: Provision only within Oracle Always Free limits (Max 2 OCPU/12GB RAM for ARM, 200GB Block Storage).
STRICT LIMIT: Use Oracle Autonomous Database (ATP) to stay within the free tier.
