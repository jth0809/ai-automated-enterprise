- 검토일: 2026-07-13
- 입력 코퍼스: `historical-candidates-v1.jsonl` 210건
- 생산 매핑: `backfill-v1.json` 120개 청구
- 고유 사용 후보: 117건
- 미선정 후보: 93건(사실 거절이 아니라 중복·경계·목표 범위 때문에 노드 중립 코퍼스에 보존)
- 노드 세트: `nodes-v1.0`
- 루브릭: `r2.0`
- 출처 코드: `NASA`, `FAA`, `UNOOSA`, `GOVINFO`, `LSA`
- 검증 기대값: 120건 모두 `OFFICIAL`

## 승인 범위와 검토 책임

사용자는 `nodes-v1.0`과 정확히 210건인 후보 코퍼스 구축 및 백필 진행을 승인했다. 이 매핑 패스에서는 각 행의 사실 메타데이터 검토와 루브릭 경계 검토를 분리해 수행했고, 두 상태가 모두 `APPROVED`인 행만 생산 JSON에 포함했다. 별도의 제3자 검토자가 참여했다고 주장하지 않는다.

## 저작권·저장 정책

생산 매핑에는 제3자 원문, 인용문, 발췌문, HTML, PDF, 이미지, 첨부파일 또는 출처 제목을 저장하지 않는다. 후보 식별자, 공식 URL 참조, 접근일, SHA-256 지문, 발행 경로, 직접 작성한 사건 제목·경계 설명만 사용한다. 원문 응답 바이트는 저장하지 않으며 가져오기 중 네트워크나 LLM 호출도 하지 않는다.

## 판정 규칙

1. 후보의 날짜와 정밀도는 그대로 복사하며 정밀도를 올리지 않는다.
2. 하나의 근거 참조는 해당 `factSummary`가 직접 지지하는 노드에만 사용한다.
3. `SETBACK`, `PROGRAM_CANCELLATION`, `ANNOUNCEMENT_ONLY`, `RETROSPECTIVE`는 상태를 올리지 않는다.
4. 부분 재사용, 지상 전용 계통, 단일 부품, 계획·계약만으로 운영 등급을 추론하지 않는다.
5. 통합 노드는 동일 캠페인과 동일 운영 구간의 직접 근거가 없으면 0을 유지한다.
6. 다섯 출처는 모두 Tier 1 공공기관이므로 현재 파생 규칙상 `OFFICIAL`이다.

## 레벨 0 유지 사유

- `P1-REUSE-LV`: 선정 근거는 1단 재사용 또는 실패 기록뿐이며, 모든 임무 필수 단계를 회수·재비행한 완전 재사용 아키텍처를 입증하지 않는다.
- `P1-TRANSPORT-INTEGRATION`: 7개 P1 요소가 하나의 책임 있는 정착 수송 캠페인에서 함께 운용된 직접 근거가 없다.
- `P2-HEALTH-AUTONOMY`: AMOS는 지상 지원 없는 영상 획득만 입증하며 자율 진단·치료·임상 운영까지 입증하지 않는다.
- `P2-SURVIVAL-INTEGRATION`: 6개 P2 요소를 같은 승무원에게 적용하며 26개월 연속 무물자보급을 관측한 기록이 없다.
- `P3-POWER`: 선정 기록은 고립형 전원, 궤도 계통 또는 지상 아날로그로서 행성 표면 정착 규모 계통 경계를 충족하지 않는다.
- `P3-THERMAL`: 직접 기록은 궤도 열제어 또는 고장 사례이며 행성 표면 주기·환경에서의 거주지 열제어를 입증하지 않는다.
- `P3-HABITAT-INTEGRATION`: CHAPEA는 지상 아날로그이고 같은 외계 표면 구간에서 건설·전력·통신·열·먼지 제어를 모두 운용하지 않았다.
- `P4-RESOURCE-INTEGRATION`: ISRU·전력·저장·소재·제조가 하나의 감사 가능한 물질·에너지 사슬로 운용된 근거가 없다.
- `P5-OPS-INTEGRATION`: ISAAC 기록은 합성 고장과 물류 시뮬레이션이며 같은 기지에서 건설·운영 자율성·물리 복구를 결합하지 않았다.
- `P6-SETTLEMENT-INTEGRATION`: 실제 정착 운영에서 시장·구속력 있는 법제·지속 금융·보험/표준이 함께 스트레스를 견딘 근거가 없다.

## 미선정과 거절 처리

93개 후보는 생산 매핑 목표 120건을 맞추고 중복 사건, 경계 밖 사례, 직접성이 약한 계획을 억지로 노드 상태로 만들지 않기 위해 미선정했다. 이 후보들은 `READY_FOR_MAPPING` 상태로 원본 코퍼스에 남아 향후 노드 또는 루브릭 개정 때 다시 검토할 수 있다. 이번 생산 JSON에는 사실 검토 또는 루브릭 검토가 `REJECTED`인 행이 없다.
